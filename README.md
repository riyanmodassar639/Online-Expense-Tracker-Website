# 💰 Expense Tracker

## Overview

Expense Tracker is a full-stack web application designed to help users manage their personal finances efficiently. Built with **React**, **Node.js**, **Express.js**, and **SQLite**, the application enables users to track income and expenses, manage transactions, monitor spending habits, and securely access their accounts through authentication.

---

## Features

* Modern and responsive UI
* User Authentication (Login & Registration)
* Add, edit, and delete transactions
* Income and Expense Tracking
* Transaction History
* Dashboard with financial summary
* File/Image Upload Support
* Subscription-based features
* Secure REST API
* Responsive design for desktop and mobile

## Installation

Clone the repository

```bash
git clone https://github.com/riyanmodassar639/expense-tracker.git
```

Navigate to the project

```bash
cd expense-tracker
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Install backend dependencies

```bash
cd ../backend
npm install
```

### Run the backend

```bash
npm start
```

### Run the frontend

```bash
cd ../frontend
npm run dev
```
